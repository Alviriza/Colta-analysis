## 📖 Tentang proyek ini

Repositori ini bukan situs portofolio biasa, dan bukan pula slide deck yang dikemas ulang sebagai web. Ia adalah **ruang kerja** — kumpulan dokumen strategi merek yang dirancang untuk dibaca, dipresentasikan, dan dieksekusi tanpa perlu terjemahan tambahan. 🛠️

Colta adalah merek aksesori produktivitas kerja yang berdiri di persimpangan tiga hal:

> 💼 **Kerja yang produktif** · 🌱 **Hidup yang sehat** · 🤍 **Estetika yang tenang**

Untuk membangun merek seperti itu di 2026, satu dokumen saja tidak cukup. Maka di sini ada empat — masing-masing berdiri sendiri sebagai bacaan, tetapi saling melengkapi sebagai sistem.

Landing page yang Anda lihat pertama berfungsi sebagai *pintu masuk* 🚪. Ia menyederhanakan empat dek menjadi empat kartu — agar siapa pun yang membuka tautan, baik tim internal, calon klien, atau investor, langsung tahu apa yang ada di balik tiap halaman, dan ke mana harus mulai membaca.

## 📚 Apa yang ada di dalam

### 📊 01 · Brand Analysis 2026
**Dokumen strategi inti.** Riset pasar aksesori produktivitas, analisis kompetitor, positioning Colta, dan kerangka kerja merek lintas dimensi — dari operasional, sumbernya, hingga keuangan. 

> 💡 Inilah dokumen yang menjawab pertanyaan paling mendasar: *kenapa Colta, kenapa sekarang, dan untuk siapa.*

🔍 Market Research · 🎯 Positioning · ⚔️ Competitor Map · 📦 Operations · 💰 Keuangan

### 🎨 02 · Brand Kit 01
**Sistem identitas visual.** Logo utama dan variasinya, palet warna inti (obsidian, porselen, abu hangat), tipografi, ikonografi, hingga panduan aplikasi untuk media sosial, banner, kemasan, apparel, dan UI aplikasi.

> 💡 Dirancang sebagai *single source of truth* untuk siapa pun yang menyentuh visual Colta — desainer internal, vendor, maupun kreator pihak ketiga.

🔷 Logo System · 🎨 Color Palette · 🔤 Typography · 📐 Iconography · 👕 Apparel · 📱 App UI

### 📱 03 · Instagram Analytics 2026
**Audit dan rencana sosial.** Analisis performa @colta.id saat ini — content pillars, anatomi feed 9-grid, format mix, heatmap waktu posting optimal, persona audiens, peta kompetitor, hingga roadmap konten 12 bulan ke depan.

> 💡 Ditutup dengan playbook hashtag, caption, dan kolaborasi kreator.

> 💡 Dari ide produk sampai pesanan B2B pertama — semuanya dalam satu dek.

📈 Trend Analysis · 🌐 Web Design · 🛒 PDP E-commerce · 📣 Paid Ads · 🤝 KOL · 💬 WA Broadcast · 🏢 B2B Channel

## 🧭 Filosofi desain

Empat prinsip yang membentuk setiap halaman di ruang kerja ini:

### 🎯 Konteks dulu, eksekusi menyusul
Sebelum menentukan warna atau menulis caption, kami memahami siapa yang membaca, keputusan apa yang harus diambil, dan kendala apa yang nyata.

### 🔬 Riset dari sumber asli
Data marketplace, performa Instagram aktual, peta kompetitor langsung. Tidak ada dataset mainan, tidak ada angka ilustratif yang menyamar sebagai fakta.

### 🧩 Sistem yang utuh
Riset → strategi → identitas → eksekusi sosial → peluncuran. Tidak ada potongan yang menggantung tanpa konteks.

### 📈 Dampak yang terukur
Setiap rekomendasi punya target, tenggat, dan saluran. Bukan *"lakukan X"*, tapi *"lakukan X melalui Y agar Z dalam waktu N."*

## 🎨 Tampilan dan teknologi

Dibangun sebagai **static site murni** — HTML, CSS, dan vanilla JavaScript. ⚡ Tidak ada framework, tidak ada build step, tidak ada dependensi runtime. Buka file di browser mana pun, ia bekerja.

| Aspek | Pilihan | Alasan |
|---|---|---|
| 🔤 **Tipografi** | Plus Jakarta Sans + JetBrains Mono | Karakter Indonesia yang modern, tanpa kesan generik |
| 🎨 **Tema** | Hitam pada putih | Fokus pada konten, bukan pada dekorasi UI |
| 📐 **Layout** | Sidebar workspace + landing modular | Setiap dek terasa seperti aplikasi, landing terasa seperti portofolio |
| ⚡ **Interaksi** | CSS-only + IntersectionObserver | Cepat, ringan, tanpa beban JavaScript besar |
| 📱 **Responsif** | Mobile-first breakpoints | Bisa dibuka dari ponsel klien tanpa zoom |
| 🚀 **Hosting** | GitHub Pages | Gratis, cepat, dan ter-versi otomatis |

> 🛡️ Tidak ada CDN eksternal selain Google Fonts. Tidak ada tracker. Tidak ada analytics tersembunyi. Yang Anda kirim ke pembaca adalah dokumen — bukan paket cookie.

## 💻 Menjalankan secara lokal

Tidak perlu Node, npm, atau toolchain apa pun. ✨ Cukup:

```bash
# 1️⃣ Clone repositori
git clone https://github.com/Alviriza/Colta-analysis.git
cd Colta-analysis

# 2️⃣ Buka langsung di browser
open index.html
# atau di Linux: xdg-open index.html
# atau di Windows: start index.html
```

⚡ Jika ingin server lokal sederhana (opsional, berguna untuk hot reload selama editing):

```bash
# 🐍 Python 3
python3 -m http.server 8000

# 📦 Node.js (jika punya npx)
npx serve .
```

Lalu buka 👉 [http://localhost:8000](http://localhost:8000)

## 🚀 Deploy ke GitHub Pages

Repositori ini sudah disiapkan untuk GitHub Pages tanpa konfigurasi tambahan:

| Langkah | Aksi |
|:---:|---|
| 1️⃣ | Buka **Settings** repositori di GitHub |
| 2️⃣ | Masuk ke menu **Pages** di sidebar kiri |
| 3️⃣ | Pada *Build and deployment*, pilih **Source:** Deploy from a branch |
| 4️⃣ | Pilih **Branch:** `main` / `root` |
| 5️⃣ | Klik **Save** ✅ |

⏱️ Tunggu 1–2 menit, situs akan tersedia di:

```
🌐 https://alviriza.github.io/Colta-analysis/
```

🔄 Setelah aktif, setiap `git push` ke `main` akan otomatis mem-publish ulang situs dalam hitungan detik.
